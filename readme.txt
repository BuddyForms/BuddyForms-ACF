== Description ==

With the BuddyForms ACF Extension you can use Advanced Custom Fields with BuddyForms.
<br>

<h4>ACF Free and Pro</h4>
Support free and Pro version of ACF out of the box. It does not matter if you use the free or the pro version of ACF.
The Plugin will detect if pro is available and loads the correct ACF field groups
<br>

<h4>ACF for the backend and BuddyForms for the front end.<h4>
ACF is the prefered choice for creating post metaboxes for the edit screen in the WordPress backend (wp-admin). It comes packed with a tons of great form elements and features.
Use all of the ACF features in the front end and combine the best of both worlds. BuddyForms ACF works with all ACF Extensions and BuddyForms Extensions.
<br>

<h4>Use ACF to create a BuddyPress Component </h4>
Create a BuddyPress Members Component or extend your groups with ACF fields.
<br>

<h4>Use ACF for all your field<h4>
With ACF enabled you build your fields once and use everywhere. Use ACF for all form elements needed and just integrate the ACF field groups or single fields in your BuddyForms forms.
You can use any ACF Field or Custom ACF Fields with BuddyForms and combine it them with BuddyForms Fields and Extensions.
<br>

<h4>Moderation for your ACF Forms</h4>
With BuddyForms Moderation you get real post submission moderation to your hands.
Let your users create and edit posts without creating the ugly 404 WordPress is struggling if a published post is set back to draft.
<br>

<H4>Use everywhere<h4>
With BuddyForms and Advanced Custom fields together you build your field sets once and use everywhere.
In the backend edit screen or in the front end via shortcodes or integrated with BuddyPress or any other BuddyForms supported plugin.

See the list of BuddyForms Extension.
See a list of ACF Extensions

combine the bower and get the most out of your create and edit forms in the front and backend seamless
<br>

== Installation ==

You can download and install the plugin using the built in WordPress plugin installer. If you download BuddyForms manually,
make sure it is uploaded to "/wp-content/plugins/".

Activate the plugin in the "Plugins" admin panel using the "Activate" link. If you're using WordPress Multisite, you can optionally activate BuddyForms Network Wide.
